# IPL-2025-BATTERS
# IPL 2025 Batters Analysis – Power BI Dashboard

##  Project Overview

This project analyzes **IPL 2025 batters performance** using **Power BI**.
The dashboard provides insights into player statistics such as runs, strike rate, boundaries, and performance comparisons.

## Objectives

* Analyze batting performance in IPL 2025
* Identify top-performing players
* Compare strike rates and run contributions
* Visualize trends using interactive dashboards

## 🛠 Tools & Technologies

* **Power BI**
* **Excel / CSV dataset**
* **Data Visualization**
* **DAX (Data Analysis Expressions)**
